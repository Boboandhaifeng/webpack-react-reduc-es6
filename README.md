# webpack-react-redux-es6
webpack3.6  react16.0  react-router4.x webpack-dev-middleware


## 简介
**从头搭建的一套 webpack react redux es6 react-router 脚手架**
**版本基本为目前最新**

**引用了 antd-design 组件库**

**开发环境 使用的 webpack-dev-middleware webpack-hot-middleware 进行热更新**

## 启动项目：
    1， npm install
    2,  npm start

## 生产环境发布包：
     npm run build
